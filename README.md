# credit-default-ml-practice
Predicting credit card default using Logistic Regression.  This project focuses on data preprocessing, EDA, model training, and evaluation as part of DS/ML learning practice.

## Project Overview
This project predicts whether a credit card customer will default on their payment using machine learning classification techniques.

## Problem Statement
Given customer financial and demographic data, the goal is to predict if the customer will default next month.

This is a binary classification problem.

## Dataset
- Credit card customer data
- Target column: default_payment_next_month
- Rows: ~30,000

## Machine Learning Workflow
- Exploratory Data Analysis (EDA)
- Data preprocessing
- Train-test split
- Logistic Regression model
- Model evaluation

## Model Used
- Logistic Regression

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
